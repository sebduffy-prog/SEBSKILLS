# raw-files

Source material that isn't part of the structured `skills/` framework.

- **`*.zip`** — Upstream skill collections (karpathy, awesome-agent-skills, skills-main, superpowers) kept for reference and re-extraction.
- **`*.skill`** — Original single-file skill packages. Their expanded, canonical versions live in `skills/ui-effects/`.

Nothing here is loaded by Claude Code automatically. Keep files here if they are reference/raw source; promote them into `skills/<category>/<name>/` when they graduate into the framework.
